# Serious CRM Readiness

Продукт можно начинать называть серьёзной CRM только когда одновременно соблюдены все блоки ниже.

## Продуктовый уровень

- dashboard реально используется как место ежедневного старта дня
- customer/deal/task/activity связаны в единую историю работы
- поиск и command palette открывают любую ключевую сущность быстро и стабильно
- импорт/экспорт/синхронизация с таблицами не ощущаются как опасный эксперимент

## Операционный уровень

- права и границы доступа не текут между организациями
- audit trail покрывает критические операции
- SLA и инцидентные процедуры существуют не только в голове разработчика
- релизы не требуют героизма и молитв

## Рыночный уровень

- пилотный клиент может запуститься без ручной сборки костылей
- продукт даёт заметное ускорение команде уже в первую неделю
- есть реальный конкурентный контур: spreadsheet superiority + daily execution + automation discipline
